# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Aybdell/portfolio.git
```

Navigate to the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

# 🏗️ Build For Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
